## Description
The loyalty program information for the user, as entered in Concur.

Loyalty programs contain a variety of information about the user's loyalty membership including:

* Vendor
* Number
* Status
* Points Total
* Segment Total
* Next Status
* Unit to Next Status
* Segments to Next Status
