## Getting Access Tokens
You can get an access token using any of the following OAuth authorization flows:

[App Center Flow][1]

[Auto-Connect Flow][2]

[Native Flow][3]

[Web Flow][4]

See [Choosing an Authorization Flow][5] for guidance on when to use each authorization flow.
